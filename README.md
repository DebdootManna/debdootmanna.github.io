# Debdoot Manna's Personal Website

This repository contains the code for Debdoot Manna's personal website, showcasing his work as a musician and developer.

## 🎵 Project Overview

This website is a retro game-inspired personal portfolio for Debdoot Manna, featuring:

- An "About Me" section
- Project highlights
- Music showcase
- Contact information
- A unique visitor counter using Supabase

## 🕹️ Features

- Retro game-inspired design with Pac-Man elements
- Responsive layout
- Animated background with random dots
- Pac-Man and Ghost animations
- Visitor counter using Supabase for data storage

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Supabase (for visitor counter)

## 📁 File Structure

- `index.html`: Main HTML file containing the website structure and inline styles
- `visitor-counter.js`: JavaScript file for the Supabase-powered visitor counter

## 🚀 Setup and Deployment

1. Clone this repository
2. Replace the Supabase URL and anon key in `visitor-counter.js` with your own Supabase project credentials
3. Ensure you have a `visitors` table in your Supabase project with columns `id` and `count`
4. Deploy the files to your preferred web hosting service

## 🔧 Customization

To customize this website for your own use:

1. Update the personal information in `index.html`
2. Replace `photo.png` with your own profile picture
3. Modify the color scheme by changing the CSS variables
4. Add or remove sections as needed

## 📊 Visitor Counter

The visitor counter uses Supabase to store and retrieve the count. Make sure to set up your Supabase project and update the credentials in `visitor-counter.js`.
